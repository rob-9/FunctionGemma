What's running under the hood

ON-DEVICE

FunctionGemma 270M

Function-calling model running via Cactus Compute (C++ FFI). Sub-250ms inference. Handles alarms, timers, weather, messaging, music, contacts.

ON-DEVICE

Qwen3 1.7B

Splits multi-intent queries ("set a timer and check weather") into sub-queries before routing each to FunctionGemma.

CLOUD (WHEN AVAILABLE)

Gemini 2.5 Flash

Handles complex queries the local model isn't confident about. Only called when signal is available.

SYNC

Offline Queue + Drain

Low-confidence offline answers are queued. When signal returns, the queue drains through cloud and corrections are surfaced.
